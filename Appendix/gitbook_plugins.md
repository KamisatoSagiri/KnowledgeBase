# GitBook 插件

|plugin name    |plugin description         |
|:-----:        |:-----:                    |
|chapter-fold   |To fold the chapter title  |

## 如何安裝插件

### 1. 在book.json中将要安装的插件名添加到plugins项中。

```json
{
    "plugins": [
        "chapter-fold"
    ]
}
```

### 2. 在SUMMARY.md文件同級目录下打开Powershell。

### 3. 输入如下指令：npm install gitbook-plugin-[plugin name]。
